# Analog Clock Project

## Description
This is a simple analog clock web application built using HTML, CSS, and JavaScript. It displays the current time in an aesthetically pleasing and easy-to-read format.

## Installation
1. Clone the repository: `git clone https://github.com/Omoha-C137/analogclock.git`
2. Open `index.html` in your preferred browser.

## Usage
To use the analog clock:
1. Simply open the `index.html` file in your browser.
2. The clock will automatically display the current time.
